Worum geht es?

CO2 Transparency ist eine Demo-Webseite, die zeigt, wie man CO2-Emissionsdaten pro Land und Unternehmen uebersichtlich darstellen kann. Die Daten sind fiktiv und dienen als Lern- und Uebungsprojekt. Der Fokus liegt auf einem klaren Aufbau, responsivem Design und sicherer Verarbeitung von Eingaben.

Was kann ich hier machen?

  • CO2-Daten in einer Tabelle ansehen.
  • Nach Land und Unternehmen filtern (Textsuche).
  • Nach Spalten sortieren (Klick auf Tabellenkopf).
  • Das Layout umschalten: lokales Menue links oder rechts.

Welche Daten werden genutzt?

Verwendet wird ein kleines, fiktives Datenset mit den Feldern: Land, Unternehmen, CO2 (Mt pro Jahr) und Jahr. Die Daten werden im Browser verarbeitet und die Tabelle wird dynamisch gerendert.

Welche Daten werden gespeichert?

Diese Demo speichert keine personenbezogenen Daten auf einem Server. Im Browser wird nur eine kleine Einstellung gespeichert: die Menue-Position (links/rechts) via localStorage, damit sich das Layout beim Seitenwechsel nicht zuruecksetzt. Filtereingaben werden nur lokal verarbeitet und nicht uebertragen.

Sicherheit

Eingaben werden so verarbeitet, dass kein injizierter Code ausgefuehrt wird. Die Tabelle wird ueber sichere DOM-Methoden aufgebaut und Inhalte werden als Text ausgegeben (textContent statt innerHTML).
